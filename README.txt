How To play?
1. Klik logo dolar di tengah halaman.
2. Klik sepuasnya.
3. Terdapat multiplier value, klik untuk melimpat-gandakan uang yang akan dihasilkan.
4. Jika ingin kaya, maka berdoalah.
5. Kehancuran adalah sebuah pilihan.
